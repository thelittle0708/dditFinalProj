**요구사항 정의서**

공략게시판 

1. 글 목록보기 
1) 모든 사용자가 볼 수 있음
2) 출력내용
  - 글번호(pk)
  - 제목
  - 작성자
  - 작성날짜

2. 글 조회 
1) 글번호로 해당 글 조회
2) 출력내용 
  - 글 번호(pk)
  - 제목
  - 글 내용
  - 작성자
  - 작성날짜

3. 글 작성
1) 모든 유저가 등록 가능
2) 입력 사항
  - 글 제목
  - 글 내용
3) 작성날짜 : 현재 날짜

4. 글 수정 
1) 글 수정은 글 작성자만 가능
2) 수정 가능 범위
  - 제목
  - 내용
3) 수정할 부분을 입력받아 일치하는 부분 수정
  - 제목 
  - 내용 
  - 날짜는 자동으로 현재 날짜

5. 글 삭제
글번호(pk)를 조회하여 삭제
